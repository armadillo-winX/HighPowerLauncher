～はじめに～

本ソフトウェアは、Windows用のアプリケーションランチャーです。
本ソフトウェアの使用によって生じたいかなる損害についても、開発者は一切責任を負いません。

○動作環境
Microsoft Windows 11/10
.NET Desktop Runtime 7.0 が必要です。

○使い方
・起動すると登録したアイテム名が一覧表示されます。
・アイテム名をダブルクリックすると起動します。

※ご注意※
・アイテムを追加する際に、アイテム名に
/ \ : * ? " < > |
の9文字は使用しないでください。
